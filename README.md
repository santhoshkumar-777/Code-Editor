# 🚀 Code Editor

A premium, cloud-inspired development environment built with React, Vite, and AI integration. This editor provides a seamless coding experience with built-in AI assistance, a SQL workbench, and deployment tools.

![Code Editor Screenshot](/public/favicon.png)

## ✨ Features

- 🤖 **AI-Powered Code Analysis**: Execute code across multiple languages with real-time feedback powered by Gemini 1.5 Flash.
- 💬 **Code Assistant**: Integrated AI chat to help with debugging, refactoring, and general coding questions.
- 📊 **SQL Workbench**: Browser-based SQLite engine with result visualization and local persistence.
- 🚢 **Docker Integration**: Generate `Dockerfile` and `docker-compose.yml` configurations with a single click.
- 🛠️ **File Explorer**: Advanced file management with support for folders and standard file operations.
- 🎹 **Mechanical Sound Effects**: Immersive mechanical keyboard sounds with multiple switch options (Blue, Brown, Red).
- 🎨 **Dynamic Themes**: Multiple built-in and customizable themes with deep indigo and glassmorphic aesthetics.
- ⏱️ **Time Travel**: Local history snapshots allowing you to restore previous versions of your code.

## 🛠️ Tech Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS
- **Editor Engine**: Monaco Editor
- **AI Engine**: Google Gemini API
- **Database**: SQL.js (SQLite WASM)
- **Styling**: Vanilla CSS + Glassmorphism

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- A Google Gemini API Key

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/code-editor.git
   cd code-editor
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure environment variables**:
   Create a `.env.local` file in the root directory:
   ```env
   VITE_GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Start the development server**:
   ```bash
   npm run dev
   ```
